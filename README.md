# TaskFlow – GUI Design (Part A)

Designed with **Claude Design** (claude.ai/design), Opus 4.8.

## Screen 1 – Login
**Prompt:** Erstelle einen Login-Screen für eine mobile ToDo-App namens „TaskFlow". Weißer, minimalistischer Hintergrund, lila Akzentfarbe (#6C63FF). Elemente: App-Logo oben links, Überschrift „Willkommen zurück", zwei Felder (E-Mail, Passwort), lila „Anmelden"-Button, Link „Noch kein Konto? Registrieren".

## Screen 2 – Aufgabenliste
**Prompt:** Erstelle einen zweiten Screen für TaskFlow: die Aufgabenliste. Oben Datum und Titel „Meine Aufgaben", Avatar-Icon. Filter-Tabs: „Alle" (aktiv, lila), „Aktiv", „Erledigt". Liste mit 5 Aufgaben als Karten mit Checkbox, Titel und farbigem Kategorie-Badge. Eine Aufgabe abgehakt und durchgestrichen. Runder lila „+"-Button unten.

## Screen 3 – Neue Aufgabe
**Prompt:** Erstelle einen dritten Screen für TaskFlow: „Neue Aufgabe". Zurück-Pfeil oben links. Felder: Aufgabenname, Fälligkeitsdatum mit Kalender-Icon, Priorität (Niedrig / Mittel / Hoch), Kategorie-Dropdown, Notizen-Textfeld. Lila „Aufgabe hinzufügen"-Button unten.
